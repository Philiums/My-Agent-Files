## 6. Comparison Articles (OpenClaw vs Other Frameworks)

### OpenClaw Vision (Design Philosophy)
- **URL:** https://github.com/openclaw/openclaw/blob/main/VISION.md
- **Description:** Explains why OpenClaw differs from other frameworks: terminal-first design, security tradeoffs, TypeScript choice, plugin architecture.
- **Why it's valuable:** Understands tradeoffs vs other agent frameworks like LangChain, AutoGPT, etc.

### Key Differentiators:
- **Local-first:** Runs on your devices, not cloud-dependent
- **Multi-channel:** WhatsApp, Telegram, Slack, Discord, 20+ platforms
- **Security-first:** Pairing codes for DM access, explicit security knobs
- **Terminal-first:** Explicit setup showing docs, auth, permissions
- **Skills ecosystem:** ClawHub for community plugins
